### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/wyakaga/coffee-shop-mobile.git
   ```

2. Check out to the project directory

```sh
cd coffee-shop-mobile
```

3. Run the development server

```sh
npm run start
```

4. Choose the app platform to develop

- Android

```sh
npm run android
```

- iOS

```sh
npm run ios
```
